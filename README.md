### Описание тестов класса BooksCollector

Тесты выполнены на pytest и покрывают все основные функции класса BooksCollector.

1. **test_add_new_book_add_two_books**
* Проверяет базовую функциональность добавления книг
* Убеждается в возможности добавления нескольких книг
* Контролирует корректность работы метода add_new_book

2. **test_add_new_book_empty_name**
* Тестирует валидацию входных данных
* Проверяет обработку пустого названия
* Контролирует соблюдение минимального ограничения длины

3. **test_add_new_book_long_name**
* Проверяет ограничение максимальной длины названия (40 символов)
* Тестирует работу валидатора длины
* Убеждается в отклонении слишком длинных названий

4. **test_add_new_book_existing_book**
* Проверяет механизм предотвращения дублирования
* Тестирует уникальность названий
* Контролирует отсутствие дубликатов при повторном добавлении

5. **test_set_book_genre_valid_genre**
* Проверяет корректность установки жанра
* Тестирует работу с допустимыми жанрами
* Контролирует правильность присвоения жанра

6. **test_set_book_genre_invalid_genre**
* Тестирует обработку несуществующих жанров
* Проверяет валидацию жанра
* Убеждается в невозможности присвоения некорректного жанра

7. **test_get_books_with_specific_genre**
* Проверяет фильтрацию по жанру
* Тестирует корректность поиска
* Контролирует правильность формирования списка

8. **test_get_books_for_children**
* Проверяет механизм определения детских книг
* Тестирует фильтрацию по возрастному рейтингу
* Контролирует корректность формирования списка

9. **test_get_books_for_children_with_age_rating**
* Проверяет исключение книг с возрастным рейтингом
* Тестирует корректность фильтрации
* Убеждается в отсутствии взрослых книг в детском списке

10. **test_add_book_in_favorites**
* Проверяет добавление в избранное
* Тестирует работу со списком избранных
* Контролирует корректность добавления

11. **test_delete_book_from_favorites**
* Проверяет удаление из избранного
* Тестирует корректность удаления
* Контролирует отсутствие книги после удаления

12. **test_add_nonexistent_book_to_favorites**
* Проверяет обработку несуществующих книг
* Тестирует защиту от некорректного добавления
* Убеждается в невозможности добавления несуществующих книг

13. **test_set_book_genre_various_genres**
* Цель: проверка установки различных жанров
* Тестируемые сценарии:
  * Установка допустимых жанров
  * Обработка недопустимых жанров
  * Проверка сохранения пустого жанра при ошибке
* Контролируемые параметры:
  * Корректность установки
  * Валидация данных
  * Обработка ошибок

14. **test_add_new_book_various_names**
* Цель: проверка различных сценариев добавления
* Тестируемые сценарии:
  * Добавление корректных названий
  * Обработка длинных названий
  * Обработка пустых названий
  * Проверка размера коллекции
* Ожидаемый результат: корректное соответствие длины коллекции

15. **test_get_books_for_children_various_genres**
* Цель: проверка фильтрации по жанрам
* Тестируемые сценарии:
  * Книги с допустимыми жанрами
  * Книги с возрастными ограничениями
* Контролируемые условия:
  * Корректность включения в детский список
  * Правильность исключения книг
  * Работа механизма фильтрации
